# Checklists

스프린트 게이트 기반 운영용 체크리스트 모음.

## 파일 목록

1. `GATE_CHECKLIST.md`
- 요구사항/기능정의/설계/스펙 게이트 및 구현 이후 단계 게이트 점검

2. `IMPLEMENTATION_TEST_CHECKLIST.md`
- 구현/테스트/보안/리뷰 완료 기준 점검

3. `SPRINT_RESULT_ANALYSIS_CHECKLIST.md`
- 결과정리 및 분석, 다음 요구사항 업데이트 점검

## 권장 사용 순서

1. 티켓 생성 시 `GATE_CHECKLIST.md` 초기화
2. 구현/테스트 단계에서 `IMPLEMENTATION_TEST_CHECKLIST.md` 실행
3. 스프린트 종료 전 `SPRINT_RESULT_ANALYSIS_CHECKLIST.md` 완료
