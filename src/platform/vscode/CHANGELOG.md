# Change Log

## [0.1.0] - 2026-02-14

### Added
- Initial release
- Task submission and monitoring via command palette and webview panel
- Agent monitoring sidebar with real-time state indicators
- Real-time updates via Server-Sent Events (SSE)
- Task detail webview with status timeline, progress bar, and log streaming
- Connection status bar item with server state display
- 7 commands for ACA interaction (submit, status, agents, stop, logs, configure, detail)
- Configurable server URL and authentication
- Support for both HTTP and HTTPS server connections
- Auto-connect on startup (configurable)
- Notification preferences for ACA events
